About the Project

Funded by the BBSRC, the BioSolr Project is a collaboration between Flax and the European Bioinformatics Institute aims “to significantly advance the state of the art with regard to indexing and querying biomedical data with freely available open source software”.

See this blog post for some more background.

Areas of Interest

BioSolr is concerned with improving the indexing and querying on biomedical data, with special regard to

  • Indexing of ontologies
  • Indexing data that is externally cross-referenced
  • Federating searches
  • Optimising search for scientific terminology, not just natural language

For more information on each of these areas, and the specific projects BioSolr has focused on, see the Documentation pages.

Demos

There are a three demo services to illustrate some of the uses of BioSolr. These are...

  • X-Joins
  • Hierarchical Faceting
  • facet.contains

Feel free to try these demos out, explore the types of functionality on offer, and please, let us know what you think!

Code

We use a GitHub repository to publish the open source software we develop to enhance and improve the use of the Apache Lucene/Solr search engine for this important sector.
